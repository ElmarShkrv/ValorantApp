# # Valorant-App
Valorant Guide is a sample Android application using [Valorant API](https://valorant-api.com/)

## Project Features 🛠️
 - MVVM with Clean Architecture
 - Flow
 - Navigation Components
 - Hilt for Dependency Injection
 - Retrofit
 - Glide
